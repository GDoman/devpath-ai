# DevPath AI

A beginner-friendly full-stack learning project with two separate apps:

- `frontend/` for the React + TypeScript + Vite app
- `backend/` for the NestJS + TypeScript app

## Project Structure

```text
devpath-ai/
  frontend/
  backend/
```

## Getting Started

Install dependencies separately in each app:

```bash
cd frontend
npm install
npm run dev
```

```bash
cd backend
npm install
npm run start:dev
```

## Why It Is Set Up This Way

This starter keeps the frontend and backend separate so each app is easier to learn and understand. It avoids auth, databases, and advanced architecture so you can focus on the basics first.
